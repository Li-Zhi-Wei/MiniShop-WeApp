# 快速开始

* 安装微信开发者工具

* 下载工程项目

* 打开``project.config``配置appID，打开``utils/config.js``配置后台网址

# 特别说明

本项目学习自七月大佬的课程，并结合项目需求增加了一些功能（如增加了邮费模块、商品套餐模块、店铺设置、用户操作埋点等），后台项目不在课程内，是我自己做的。
